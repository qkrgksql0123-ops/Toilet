01 / 앱 소개

왜 볼일봐인가?

🗺️
주변 화장실 지도
행정안전부 공공데이터 API로 전국 공중화장실 위치를 Google Maps 핀으로 표시
MVP
⭐
청결도 별점 / 리뷰
사용자가 직접 청결도를 평가하고 후기를 남겨 다른 사용자에게 실시간으로 공유
MVP
❤️
즐겨찾기
자주 가는 화장실을 즐겨찾기에 저장해서 빠르게 접근
MVP
🔐
비번 공유
잠긴 화장실의 비밀번호를 커뮤니티로 공유. 좋아요로 신뢰도를 표시하는 차별화 기능
MVP · 독창적
02 / ADR

기술 결정 이유

ADR-0000

앱 주제 선택

공중화장실 지도 앱 — 실생활 필요, 공공데이터 활용, 비번 공유라는 독창적 기능

볼일봐
ADR-0001

프론트엔드 플랫폼

한 코드로 iOS/Android 동시 지원. Google Maps 공식 Flutter 패키지 존재. Hot Reload로 빠른 개발

Flutter 3.x
ADR-0002

상태관리

Provider의 공식 후속작. GPS 스트림과 Firestore 실시간 데이터를 StreamProvider로 자연스럽게 연동

Riverpod 2.x
ADR-0003

백엔드 & 데이터베이스

서버 코드 불필요. 실시간 동기화로 리뷰/비번이 즉시 반영. Flutter 공식 지원(FlutterFire)

Firebase
ADR-0004

지도 API

Flutter 공식 패키지 존재. 카카오/네이버는 비공식이라 연동 복잡. 공공데이터 좌표와 바로 연동 가능

Google Maps
